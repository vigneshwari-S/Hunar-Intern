Projects

Project 1: Simple Tip Calculator

This project is a simple tip calculator that helps users calculate the appropriate tip amount based on their bill total and desired tip percentage. Users can input the bill amount and select a tip percentage (15%, 20%, etc.), and the calculator will display the total amount including tip, as well as the tip amount separately. The user-friendly interface makes it easy for anyone to use, whether dining out or sharing a meal.

Project 2: Cake Booking Website

This project is a cake booking website that allows customers to browse a variety of cake designs and flavors. Users can select their desired cake, customize it with personal messages, and choose a delivery date. The website features an easy-to-use interface with an interactive gallery, a straightforward booking process, and payment options to ensure a seamless customer experience. Perfect for birthdays, weddings, or any special occasion!
